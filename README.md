# Introduction to Machine Learning Safety (MLS)

This repository contains coursework, exercises, reports, and practical implementations for the **Introduction to Machine Learning Safety** course at **Otto von Guericke University Magdeburg (OVGU)**.

## Course Overview

Machine learning systems behave fundamentally differently from classical software systems. This course explores the safety implications of data-driven AI systems, especially in safety-critical domains such as autonomous driving.

## Repository Structure

- `mls-ex3.ipynb` - Deep learning pedestrian detection experiments
- `mls-ex5.ipynb` - ML safety, calibration, and backdoor attack experiments
- `mls-ex5.md` - Written report for Exercise Sheet 5
